Ubuntu16.04-matplotlib解决中文乱码

你好，欢迎来到我的个人技术博客！

解决matplotlib显示中文问题

  • import matplotlib.pyplot as plt
  • plt.rcParams['font.sans-serif'] = ['Microsoft YaHei'] # 指定默认字体
  • plt.rcParams['axes.unicode_minus'] = False # 解决保存图像是负号'-'显示为方块的问题

    • 上面的操作做完之后需要把微软雅黑字体msyh.ttf放到ubuntu里你的matplotlib所在的对应字体配置下:

    比如我的路径是~/anaconda3/pkgs/matplotlib-2.0.2-np112py36_0/lib/python3.6/site-packages/matplotlib/mpl-data/fonts/ttf/

    修改字体的配置文件以我自己的为例~/anaconda3/pkgs/matplotlib-2.0.2-np112py36_0/lib/python3.6/site-packages/matplotlib/mpl-data/matplotlibrc

    要修改的地方就2个：font.family : sans-serif 注释打开； font.sans-serif : Microsoft YaHei,DejaVu Sans, Bitstream Vera Sans, Lucida Grande, Verdana, Geneva, Lucid, Arial, Helvetica, Avant Garde,sans-serif
